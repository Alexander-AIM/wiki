# Slots machine 💁‍♂🎰


::: details
Acest joc se poate regăși si sub denumirea de **Pacanele**
:::

Pentru a te așeza la un aparat, apasă tasta `F` în cercul din față aparatului dorit, selectează miza, iar apoi începe să joci.
